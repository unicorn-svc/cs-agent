# AI 에이전트 요구사항 시나리오

> 서비스 목적: 고객이 제품에 대해 질문하면 FAQ 지식베이스에서 자동으로 답변을 찾아 응답하고,
> 답변이 없는 경우 상담원에게 연결하는 에이전트
>
> 선택된 버전: 버전 3 (관점: 비용 절감)
> 생성일: 2026-03-26

---

## 버전 3 (관점: 비용 절감)

### 1. 서비스 개요

- **서비스명**: 고객센터 운영 비용 최적화 에이전트
- **서비스 유형**: Agent
- **서비스 목적**: FAQ 자동 응답으로 상담원 투입 건수를 최소화하고, 처리 건수 대비 운영 비용을
  지속적으로 낮추며 고객센터 규모를 효율적으로 유지하는 비용 절감 에이전트 구현
- **대상 사용자**: 고객센터 운영 관리자, 경영진, 비용 관리 담당팀

---

### 2. 사용자 시나리오

**현재 상태(As-Is)**
- 상담원 1인 월 인건비 약 280만 원, 10명 운영 시 월 고객센터 운영 비용 2,800만 원 이상
- 전체 문의의 65%가 동일한 FAQ 유형임에도 상담원이 1:1로 응대하여 비용 낭비 발생
- 성수기(연말·프로모션) 문의 급증 시 임시 상담원 추가 채용으로 비용 급등

**목표 상태(To-Be)**
- 반복성 FAQ 문의 자동 처리로 상담원 필요 인원 최소화
- 에이전트가 처리한 건수와 절감 비용을 실시간 집계하여 운영 관리자에게 보고
- 성수기 문의 급증에도 에이전트가 자동 확장 처리하여 추가 인력 없이 대응

**기대 효과**
- 상담원 필요 인원 10명 → 4명으로 감소(60% 절감)
- 월 고객센터 운영 비용 2,800만 원 → 1,120만 원으로 절감(연간 약 2억 원 절감)
- 성수기 임시 채용 비용 완전 제거

---

### 3. 에이전트 역할 및 행동

**역할 정의**
최대한 많은 문의를 자동 처리하여 상담원 투입을 최소화하고,
처리 결과와 비용 절감 현황을 실시간으로 집계·보고하는 운영 효율화 에이전트

**단계별 행동**

1. 사용자가 질문을 입력하면
   - 입력: 고객 질문 텍스트, 문의 채널
   - 처리: 자동 처리 가능 여부 판단(FAQ 범위 내 여부, 복잡도 평가)
   - 출력: 처리 유형 분류(자동 처리 가능 / 상담원 필수)

2. 자동 처리 가능 건으로 분류되면 FAQ 지식베이스에서 답변을 검색하면
   - 입력: 질문 텍스트, FAQ 저장소
   - 처리: 의미 기반 자료 검색으로 최적 답변 추출
   - 출력: 답변 텍스트, 처리 비용 집계 데이터(1건 처리 완료)

3. 답변을 전달하고 처리 비용 절감 실적을 집계하면
   - 입력: 처리 완료 건 데이터
   - 처리: 자동 처리 1건당 절감 비용 계산 및 누적 집계
   - 출력: 실시간 절감 비용 업데이트, 일별 처리 현황 보고서

4. 상담원 필수 건으로 분류되면
   - 입력: 질문 내용, 복잡도 분류
   - 처리: 복잡도·긴급도에 따라 상담원 배정 우선순위 결정
   - 출력: 우선순위 부여된 상담원 대기열 등록

**예외 처리**

| 상황 | 대응 방법 |
|------|-----------|
| 자동 처리율이 목표 이하(60% 미만)로 하락 | FAQ 저장소 갱신 필요 알림 및 미매칭 질문 유형 분석 보고 |
| 문의 급증(평시 대비 3배 이상) | 자동 처리 우선 최대 확장, 처리 불가 건만 상담원 배정 |
| 반복 오답 발생 | 해당 FAQ 항목 자동 비활성화 및 검토 요청 알림 |

---

### 4. 워크플로우 설계

**입력 항목**
- 고객 질문 텍스트(필수)
- 문의 채널 구분
- 문의 접수 시각

**출력 항목**
- 자동 답변 또는 상담원 이관 처리
- 건별 처리 비용 집계 데이터
- 일별/월별 절감 비용 보고서

**분기 조건**

```
만약 FAQ 검색 관련도 ≥ 75점 AND 복잡도 낮음이면 → 자동 처리 단계로 진행
만약 FAQ 검색 관련도 < 75점 OR 복잡도 높음이면 → 상담원 필수 배정 단계로 전환
만약 동시 문의 수가 평시 3배 초과이면 → 자동 처리 최대 확장 모드 전환
만약 자동 처리율 < 60%이면 → FAQ 저장소 갱신 필요 알림 발송
```

**반복 처리**
- 일 단위로 전체 처리 건수와 절감 비용 자동 집계 및 보고서 생성
- 월 단위로 FAQ 항목 최적화 권고안 자동 생성

### 5. 외부 도구 및 데이터 소스

**필요한 도구/서비스 목록**

| 기능 | 용도 | 예상 입출력 |
|------|------|------------|
| FAQ 지식베이스 의미 검색 | 자동 처리 가능 건 최대화를 위한 FAQ 탐색 | 입력: 질문 텍스트 / 출력: FAQ 항목·관련도·복잡도 점수 |
| 처리 건수 및 비용 집계 | 건별 처리 비용 계산 및 누적 절감 금액 산출 | 입력: 처리 유형·건수 / 출력: 절감 비용 집계액 |
| 운영 보고서 자동 생성 | 일별·월별 처리 현황 및 절감 비용 보고서 생성 | 입력: 집계 데이터 / 출력: 보고서 파일 |
| 상담원 대기열 관리 | 상담원 필수 건의 우선순위 배정 및 관리 | 입력: 건별 복잡도·긴급도 / 출력: 배정 완료 여부 |
| 알림 발송 | 운영 이상 지표 감지 시 관리자 알림 | 입력: 이상 지표 내용 / 출력: 발송 성공 여부 |

**자료 저장소**
- FAQ 문서 저장소: 제품별 FAQ 항목, 처리율 최적화를 위한 지속 갱신 관리
- 처리 이력 저장소: 건별 처리 유형·소요 시간·비용 기록(월별 집계용)
- 절감 비용 집계 저장소: 일별·월별·연별 누적 절감 금액 보관

**외부 연동**
- 상담원 업무 시스템: 복잡도·긴급도 기반 자동 배정 연동
- 경영진 보고 시스템: 월별 절감 비용 보고서 자동 전달

**커스텀 개발 필요 여부**
- [커스텀 개발 필요] 비용 절감 집계 및 보고 모듈 — 처리 건수와 절감 비용을 실시간 집계하고
  경영진 보고 형식에 맞는 보고서를 자동 생성하는 전용 모듈 필요
  (입력: 일별 처리 건수·유형 / 출력: 절감 비용 보고서 파일)
- [커스텀 개발 필요] 상담원 업무 시스템 연동 모듈 — 복잡도·긴급도 기반으로 상담원에게
  자동 배정하고 배정 현황을 실시간 추적하는 연동 모듈 필요
  (입력: 문의 건 복잡도·긴급도 / 출력: 배정된 상담원 ID·예상 처리 시각)

### 6. AI 지시사항 가이드

**AI 역할과 성격**
- 역할: 비용 효율을 최우선으로 하는 자동 처리 전문 에이전트
- 성격: 신속·간결하며, 불필요한 대화 없이 핵심 답변만 제공
- 어조: 정중하고 간결한 존댓말

**응답 형식**
- 언어: 한국어 기본
- 길이: 200자 이내(최대한 간결)
- 형식: 핵심 답변만 제공, 부가 설명 최소화
- 이관 안내 시 대기 시간 예상치 포함(예: "상담원 연결까지 약 5분 소요 예정")

**금지 사항**
- 불필요한 안부·감사 인사 금지(답변 효율 저하 방지)
- FAQ 범위 외 추측 답변 금지
- 상담원 배정 없이 고객을 장시간 대기시키는 행위 금지

---

### 7. 품질 요구사항

**응답 속도**
- 자동 답변: 질문 접수 후 8초 이내(빠른 처리가 비용 효율에 직결)
- 상담원 이관 등록: 20초 이내

**정확도**
- FAQ 자동 처리율: 월 평균 70% 이상 유지
- 오답률: 5% 이하(오답은 상담원 재처리 비용 발생)

**비용 관리(핵심 지표)**
- 자동 처리 1건당 목표 비용: 상담원 처리 대비 90% 절감
- 월간 자동 처리 건수 목표: 전체 문의의 70% 이상
- 처리율이 목표 미달 시 FAQ 저장소 갱신 우선 조치

**보안**
- 고객 개인정보 처리 이력 저장 금지
- 집계 데이터는 개인 식별 불가 형태로만 보관

**운영 관리**
- 자동 처리율 일별 모니터링, 60% 미만 시 즉시 알림
- 월 1회 FAQ 항목 최적화 권고안 자동 생성

### 8. 검증 시나리오

**정상 케이스**

| 케이스 | 입력 | 기대 결과 |
|--------|------|-----------|
| 케이스 1 | 하루 1,000건 문의 접수(평시 수준) | 700건 이상 자동 처리 완료(70%), 처리 이력 및 절감 비용 집계 완료 |
| 케이스 2 | "충전 케이블 연결 방법" 질문 | 8초 이내 간결 답변 제공(200자 이내), 1건 절감 비용 집계 |
| 케이스 3 | 연말 성수기 하루 3,000건 문의 급증 | 추가 상담원 없이 자동 처리 최대 확장, 처리율 65% 이상 유지 |

**예외 케이스**

| 케이스 | 입력 | 기대 결과 |
|--------|------|-----------|

| 예외 2 | 특정 FAQ 항목에서 오답 3회 연속 발생 | 해당 항목 자동 비활성화, 검토 요청 알림 발송 |
